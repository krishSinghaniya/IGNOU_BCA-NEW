# IGNOU BCA_NEW Study Resources

A student-maintained GitHub repository for **BCA students** to access **Semester books, PYQs, Notes etc**.

---

## Purpose

* Share **BCA semester-wise books / PDFs**
* Provide **Term-End Exam (TEE) PYQs**
* Provide **Other Learning Resources / Notes**
* Help current and future IGNOU BCA students in exam preparation

---

Each semester may include:

* Books
* PYQs
* Recommended video tutorial
* Notes (if available)
